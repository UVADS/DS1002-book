You’ve just seen an example of recursion!!!

Recursion in Python occurs when a function calls itself during its execution. This programming technique is useful for solving problems that can be broken down into smaller, repetitive tasks.

With each recursive call, the problem is simplified into a smaller part, and the recursion continues until it reaches a base case, where no further recursive calls are needed.

  1. Use recursion, as we have just seen, to create a function that, given an input number, it returns its factorial. For example, given a 4, it should give: 4! = 4 × 3 × 2 × 1 = 24. Test the function.

def factorial(a):
    if a == 0:
        return 1
    else:
        return a * factorial(a - 1)  # Recursive call to f itself

print(factorial(3))
print(factorial(8))

  1. (More difficult) Create a function that creates a Fibonacci sequence. A Fibonacci sequence is a series of numbers where each number in the sequence is the sum of the two previous numbers. The sequence starts with 0 and 1, and then continues with 1, 2, 3, 5, 8, 13, 21, and so on.

def fibonacci(n):
    if n == 0 or n==1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
